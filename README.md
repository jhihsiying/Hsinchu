# 在風城的故事
My Hometown, which is a windy city, Hsinchu.<br>
Demo: [https://hsinchu.jsy.tw](https://hsinchu.jsy.tw)

## 風城：序
寫在故事最前篇：簡單介紹新竹的地理位置。

## 故事大綱
每一段故事都蘊含精彩的人事物：以 Time Line 方式呈現荷西時代到民國的簡單介紹。

## 提味小品
點綴故事的彩蛋：點擊圖片即可看到每一個新竹美食的說明。

## 熱門篇章
最有人氣的十二段劇情：點擊圖片即可看到每一個新竹景點的說明。
